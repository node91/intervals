# Intervals Tray
- Create API key on `intervals.icu` and add it to `settings.json`
- Either run `intervals.py` or `intervals.exe`
- Or build with `pyinstaller --onefile --windowed --icon=intervals.ico intervals.py`
  - Python version `3.13.9`
  - Requirements: `pip install wxPython requests pillow`
- Put `intervals.exe`, `intervals.ico` and `settings.json` in the same folder for everything to work correctly
